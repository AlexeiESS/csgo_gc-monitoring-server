# csgo_gc-monitoring-server
A simple plugin for the CS:GO server for monitoring online activity on the server.
This plugin was developed for the CSGO_GC community, but can be distributed for public use. 
The plugin's concept is simple: it collects certain data on the server and sends it in JSON format to a webhook you specify. 
Important note: the plugin requires the SteamWorks extension. 
The minimum SourceMod version required is 1.12.
